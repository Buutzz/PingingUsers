
# Pinging Users

Plugin for MyBB forum engine - allows to ping users from the threads sending them Private Message informing of the ping with link to the post.

## 🔔 Usage

```
@[user_name]
```
Inside the post.

- It is secure to use inside the BB codes
- Pings users only once from single post
- After edit do not ping the same users twice

## ⚙️ Customization

In settings you can customize the subject and the message body.
